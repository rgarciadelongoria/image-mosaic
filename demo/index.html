image-mosaic

Web Component to display a mosaic of images

With this component you can display from 1 to 4 mosaic of images.

The default appearance of each mosaic depends on the number of images.

                
<image-mosaic
image1="https://picsum.photos/id/152/400/400"
image2="https://picsum.photos/id/159/400/400"
image3="https://picsum.photos/id/175/400/400"
image4="https://picsum.photos/id/237/400/400">
</image-mosaic>

Apply styles to each image directly from the url

One of the main drawbacks of using mosaics is the need to have images perfectly framed for each mosaic.

You can apply css styles directly through the styles parameter of the URL of each image, without the need to modify the HTML code

                
<image-mosaic
image1="https://picsum.photos/id/152/400/400">
</image-mosaic>

<image-mosaic
image1="https://picsum.photos/id/152/400/400&styles="background-position: 25px 50%;">
</image-mosaic>

<image-mosaic
image1="https://picsum.photos/id/152/400/400&styles=background-size: 300%;">
</image-mosaic>

Use a custom mosaic by modifying the CSS clip-path

One of the main drawbacks of using mosaics is the need to have images perfectly framed for each mosaic.

You can also remove the base styles of the mosaic with the clear-styles property.

                
<image-mosaic
    image1="https://picsum.photos/id/152/400/400?styles=clip-path: polygon(50% 50%, 0 100%, 100% 100%);"
    image2="https://picsum.photos/id/159/400/400?styles=clip-path: polygon(50% 50%, 100% 0, 100% 100%);"
    image3="https://picsum.photos/id/175/400/400?styles=clip-path: polygon(50% 50%, 100% 0, 0 0);"
    image4="https://picsum.photos/id/237/400/400?styles=clip-path: polygon(50% 50%, 0 100%, 0 0);">
</image-mosaic>


<image-mosaic
    image1="https://picsum.photos/id/152/400/400?styles=clip-path: polygon(0 0, 20% 0, 80% 100%, 0% 100%);"
    image2="https://picsum.photos/id/159/400/400?styles=clip-path: polygon(100% 0, 20% 0, 80% 100%, 100% 100%);"
    image3="https://picsum.photos/id/175/400/400?styles=clip-path: polygon(10% 10%, 10% 90%, 25% 90%, 25% 25%, 75% 25%, 75% 75%, 25% 75%, 25% 90%, 90% 90%, 90% 10%);>
</image-mosaic>
