image-mosaic

Web Component to display a mosaic of images

With this component you can display from 1 to 4 mosaic of images.

The default appearance of each mosaic depends on the number of images.

                    
<image-mosaic
    image1="./img/image1.jpg"
    image2="./img/image2.jpg"
    image3="./img/image3.jpg"
    image4="./img/image4.jpg">
</image-mosaic>

Apply styles to each image directly from the url

One of the main drawbacks of using mosaics is the need to have images perfectly framed for each mosaic.

You can apply css styles directly through the styles parameter of the URL of each image, without the need to modify the HTML code

                    
<image-mosaic
    image1="./img/image1.jpg">
</image-mosaic>

<image-mosaic
    image1="./img/image1.jpg&styles="background-position: 25px 50%;">
</image-mosaic>

<image-mosaic
    image1="./img/image1.jpg&styles=background-size: 300%;">
</image-mosaic>
                    
                
                    
<image-mosaic
    image1="./img/image1.jpg"
    image2="./img/image2.jpg"
    image3="./img/image3.jpg"
    image4="./img/image4.jpg">
</image-mosaic>

<image-mosaic
    image1="./img/image1.jpg?styles=background-position: center 140px;background-size: 250%;"
    image2="./img/image2.jpg?styles=background-position: -45px -45px;"
    image3="./img/image3.jpg"
    image4="./img/image4.jpg?styles=background-position: 15px 25px;">
</image-mosaic>

Use a custom mosaic by modifying the CSS clip-path

One of the main drawbacks of using mosaics is the need to have images perfectly framed for each mosaic.

You can also remove the base styles of the mosaic with the clear-styles property.

                    
<image-mosaic
    image1="./img/image1.jpg?styles=clip-path: polygon(50% 50%, 0 100%, 100% 100%);"
    image2="./img/image2.jpg?styles=clip-path: polygon(50% 50%, 100% 0, 100% 100%);"
    image3="./img/image3.jpg?styles=clip-path: polygon(50% 50%, 100% 0, 0 0);"
    image4="./img/image4.jpg?styles=clip-path: polygon(50% 50%, 0 100%, 0 0);">
</image-mosaic>
